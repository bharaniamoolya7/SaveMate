💰 SaveMate – Personal Financial Tracker

SaveMate is a professional, high-fidelity personal finance management platform designed to help users track their monthly spending power, manage category-based budgets, and stay on top of upcoming bills.

🧱 Technical Stack
Frontend

Built using React (with Vite).
It features a premium, responsive dark-themed UI with smooth animations, interactive charts using Recharts, and a card-based layout for a modern experience.

Backend

Powered by Spring Boot, which provides a robust REST API.
It includes Spring Security with JWT (JSON Web Token) for secure user authentication and session management.

Database

The project is architected to use MySQL as the primary relational database for storing user profiles, transaction history, and custom budget categories.

🌍 Deployment

The frontend is deployed on Netlify. For simplicity in the live demonstration, it currently utilizes a built-in Mock API service that simulates the database using browser local storage. This ensures the site works instantly for anyone with the link.

The entire backend folder structure and MySQL configurations are preserved in the Git repository for future full-scale deployment.

🎯 Project Goal

SaveMate demonstrates real-world full-stack development by combining modern frontend design, secure backend architecture, and relational database management into a single financial tracking solution suitable for both academic and portfolio showcase.
